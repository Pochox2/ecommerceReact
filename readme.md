## Dependencias

- Bootstrap [Version:"^5.2.3"] Es una biblioteca multiplataforma o conjunto de herramientas de código abierto para diseño de sitios y aplicaciones web
- Firebase [Version:"^9.17.1"] Ofrece documentación detallada y SDK multiplataforma que te ayudarán a compilar y lanzar apps en Android, iOS, la Web, C++ y Unity.
- React: [Version:"^18.2.0"] Es una biblioteca de JavaScript declarativa, eficiente y flexible para construir interfaces de usuario.
- React-bootstrap [Version:"^2.7.2"] Biblioteca de bootstrap con muchas funciones en React.
- React-dom ["^18.2.0"] Paquete que proporciona métodos específicos del DOM que pueden ser utilizados en el nivel más alto de tu aplicación como una vía de escape del modelo de React si así lo necesitas.
- React-loader-spinner ["^5.3.4"] Biblioteca de spinners para componentes.
- React-router-dom ["^6.8.1"] Biblioteca que nos permite navegar por la aplicacion.
- React-toastify ["^9.1.1"] Biblioteca de notificaciones personalizadas.

## Sitio Web
-Pagina principal: En la pagina principal podras ver todos los productos y podras navegar entre dos categorias que son Bebidas y Ropa.

-Navbar: Este elemento nos permite navegar hacia las distintas categorias como asi tambien volver al Inicio. A la derecha de las categorias estara ubicado el carro de compras que te permitira navegar hacia el mismo para poder ver los productos que ya seleccionaste.

-Items: Cada producto esta en una card distinta, nos proporciona informacion breve pero si pulsas el boton detalle te permitira navegar hacia cada producto con su respectiva informacion y podras agregarlo al carro,

## Firebase
El sitio web usara firebase como base de datos. En firebase encontraras dos colecciones:
-Productos: En esta estaran todos los productos que estan en la tienda con sus respectivos datos.
-Ordenes: En esta se iran agregando todas las ordenes de compra de los clientes.
